# トランプ・ダンジョン (Trump Dungeon)

## 概要

「トランプ・ダンジョン」は、ラビダンジョンでの生き残りをかけた、1人遊び用のRPG風カードゲームです。
山札がすべてなくなるまで生き延びることができれば、ゲームクリアとなります。

このゲームは、単一のHTMLファイルのみで動作し、特別な準備なしにブラウザで手軽に遊ぶことができます。

## 遊び方

1.  `trump_dungeon.html` ファイルをお使いのウェブブラウザで開きます。
2.  タイトル画面の「スタート」ボタンを押して、ゲームを開始します。

### ゲームの基本操作

-   **カードの選択**: 画面に提示される4枚の「ダンジョンカード」から1枚をクリックして選択します。選択したカードの種類に応じて、様々なイベントが発生します。
-   **マリガン**: どうしても手札が気に入らない場合、「マリガン」ボタンを押すことで手札をすべて引き直すことができます。ただし、ペナルティとして**HPが半分**になります。
-   **設定**: タイトル画面の「設定」ボタンから、以下の項目を変更・確認できます。
    -   **背景色の変更**: ゲーム全体の背景色をお好みの色に変更できます。
    -   **ルール**: ゲームのルールをいつでも確認できます。

### カードの種類と効果

| カードの種類        | 効果                                                                              |
| ------------------ | --------------------------------------------------------------------------------- |
| **♠/♣ (敵)**       | カードの数字が攻撃力となります。武器を装備していない、または武器の攻撃力より敵が強い場合、その差分のダメージを受けます。戦闘後、武器は必ず壊れます。 |
| **♥ (回復)**       | カードの数字分、HPが回復します。                                                    |
| **♦ (武器)**       | 装備すると、敵からのダメージを攻撃力の分だけ防ぐことができます。                    |
| **🃏 (ジョーカー)**   | このカードを選ぶと、場（手札）にある他のカードがすべて消え去り、新しいカードが4枚引かれます。 |
| **詛 (呪いのカード)** | このカードを選ぶと呪われ、**HPが5減少**します。非常に危険なカードです。            |


## 工夫した点

-   **手軽さ**: ゲームのすべてが単一のHTMLファイルにまとめられています。これにより、ダウンロードしてすぐに、どのPCでもブラウザさえあれば遊ぶことができます。
-   **画面遷移システム**: タイトル、設定、ゲーム本編、ゲームオーバーといった各画面を、ページ遷移なしでスムーズに切り替えるシステムをJavaScriptで実装しました。
-   **戦略的な難易度**: ゲームの難易度を上げるため、以下のような戦略的要素を加えています。
    -   戦闘ごとに必ず壊れる武器。
    -   HP半分という大きなコストを要求するマリガン機能。
    -   選択すると大きなペナルティを受ける「呪いのカード」。
-   **UIのカスタマイズ性**: 設定画面から背景色を自由に変更できる機能を搭載し、プレイヤーが自分好みの見た目でゲームを楽しめるようにしました。
